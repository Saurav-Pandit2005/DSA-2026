** Introduction to ArrayList
--------------------------------

-> An ArrayList in Java is a resizable array from the java.util package that can dynamically grow or shrink as elements are added or removed, unlike normal arrays which have a fixed size. 

-> It stores elements in insertion order, allows duplicates, and provides methods like add(), get(), set(), and remove() for easy manipulation. 

-> ArrayList is a part of Java Collection Framework.

    * Key Features of ArrayList
    -----------------------------

        -> Dynamic resizing: Automatically expands or shrinks when elements are added or removed.

        -> Index-based access: Elements can be accessed directly using indices, similar to arrays.

        -> Duplicates allowed: Unlike sets, ArrayList permits duplicate elements.

        -> Insertion order maintained: Elements remain in the order they were added.


    * Array v/s ArrayList
    -----------------------

             Array                                ArrayList

        -> fixed size                           -> dynamic size  

        -> primitive data types                 -> primitive data types can't
            can be stored                           be stored directly

** Operations on ArrayList
-----------------------------

->  The most common operations on an ArrayList include adding, accessing, updating, removing, iterating, and checking its size. These operations make ArrayList flexible compared to arrays, since it can grow and shrink dynamically.

    * Creating a ArrayList
    --------------------------

        ArrayList<Integer> list1 = new ArrayList<>();

    * Operations
    ---------------

        Add Element                 O(1)
        Get Element                 O(1)
        Remove Element              O(n)
        Set Element at Index        O(n)
        Contains Element            O(n)

    * Examples :
    -------------------

         // Creating a ArrayList
        ArrayList<Integer> list1 = new ArrayList<>();

        // Add Element      O(1)

        list1.add(20);
        list1.add(200);
        list1.add(2000);

        list1.add(3,20000);     // O(n)

        System.out.println(list1);

        // Get Element      O(1)

        System.out.println(list1.get(2));

        // Remove Element   O(n)

        System.out.println(list1.remove(1));
        System.out.println(list1);
        
        // Set Element on index     O(n)

        System.out.println(list1.set(1, 200));
        System.out.println(list1);

        // Contains Element

        System.out.println(list1.contains(2000));

------------------------

** Sorting an ArrayList
----------------------------

-> Check Q4.java

-> To sort an ArrayList in Java, you typically use the Collections.sort() method from the java.util package.

-> This lets you sort elements in ascending order by default, and you can customize sorting with comparators for descending or custom rules.


    * Key Points
    --------------

        -> Use Collections.sort(list) for ascending order.

        -> Use Collections.sort(list, Collections.reverseOrder()) for descending order.

        -> Use Comparator for custom sorting logic.

        -> Sorting modifies the original list (in-place).

    * Example :
    -------------

         ArrayList<Integer> list = new ArrayList<>();

        list.add(2);
        list.add(5);
        list.add(9);
        list.add(3);
        list.add(6);

        System.out.println(list);

        // Ascending Order

        Collections.sort(list);
        System.out.println(list);

        // Descending Order

        Collections.sort(list, Collections.reverseOrder());
        System.out.println(list);

** ArrayList (Implementation in java)
------------------------------------------

    * Internal Implementation of ArrayList
    ----------------------------------------

    1) Backed by an Array

        -> Internally, ArrayList uses a dynamic array (Object[] elementData) to store elements.

        -> When you add elements and the array is full, it creates a new larger array and copies the old elements into it.

    2) Default Capacity

        -> When you create new ArrayList(), it starts with a default capacity (usually 10).

        -> If you specify new ArrayList(20), it starts with capacity 20.

    3) Resizing Mechanism

        -> When capacity is exceeded, the array grows by about 50% of its current size.

        -> Example: If capacity = 10 and you add the 11th element, it resizes to 15.

    4) Index-based Access

        -> Since it’s backed by an array, accessing elements (get(index)) is O(1) (constant time).

    5) Insertion & Removal

        -> Adding at the end is usually O(1) (amortized).

        -> Adding/removing in the middle requires shifting elements → O(n).

** Multi-dimensional ArrayList (2D ArrayList)
-----------------------------------------------

-> A multi‑dimensional ArrayList in Java simply means an ArrayList that contains other ArrayLists as its elements.

-> Since Java doesn’t have true multi‑dimensional lists built in, we simulate them by nesting collections.

-> Declaration:

    2D → ArrayList<ArrayList<Type>>


    * Example :
    ----------------

         // Main list
        ArrayList<ArrayList<Integer>> list = new ArrayList<>();

        // List1
        ArrayList<Integer> list1 = new ArrayList<>();

        list1.add(1);
        list1.add(2);
        list1.add(3);
        list1.add(4);
        list1.add(5);

        // List2
        ArrayList<Integer> list2 = new ArrayList<>();

        list2.add(2);
        list2.add(4);
        list2.add(6);
        list2.add(8);
        list2.add(10);

        // List3
        ArrayList<Integer> list3 = new ArrayList<>();

        list3.add(3);
        list3.add(6);
        list3.add(9);
        list3.add(12);
        list3.add(15);

        // add list1, list2, list3 in Main List

        list.add(list1);
        list.add(list2);
        list.add(list3);

        System.out.println(list);

    
    * Output :
    -------------

        [[1, 2, 3, 4, 5], [2, 4, 6, 8, 10], [3, 6, 9, 12, 15]]
