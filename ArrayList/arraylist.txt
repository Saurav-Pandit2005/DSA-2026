** Introduction to ArrayList
--------------------------------

-> An ArrayList in Java is a resizable array from the java.util package that can dynamically grow or shrink as elements are added or removed, unlike normal arrays which have a fixed size. 

-> It stores elements in insertion order, allows duplicates, and provides methods like add(), get(), set(), and remove() for easy manipulation. 

-> ArrayList is a part of Java Collection Framework.

    * Key Features of ArrayList
    -----------------------------

        -> Dynamic resizing: Automatically expands or shrinks when elements are added or removed.

        -> Index-based access: Elements can be accessed directly using indices, similar to arrays.

        -> Duplicates allowed: Unlike sets, ArrayList permits duplicate elements.

        -> Insertion order maintained: Elements remain in the order they were added.


    * Array v/s ArrayList
    -----------------------

             Array                                ArrayList

        -> fixed size                           -> dynamic size  

        -> primitive data types                 -> primitive data types can't
            can be stored                           be stored directly

** Operations on ArrayList
-----------------------------

->  The most common operations on an ArrayList include adding, accessing, updating, removing, iterating, and checking its size. These operations make ArrayList flexible compared to arrays, since it can grow and shrink dynamically.

    * Creating a ArrayList
    --------------------------

        ArrayList<Integer> list1 = new ArrayList<>();

    * Operations
    ---------------

        Add Element                 O(1)
        Get Element                 O(1)
        Remove Element              O(n)
        Set Element at Index        O(n)
        Contains Element            O(n)

    * Examples :
    -------------------

         // Creating a ArrayList
        ArrayList<Integer> list1 = new ArrayList<>();

        // Add Element      O(1)

        list1.add(20);
        list1.add(200);
        list1.add(2000);

        list1.add(3,20000);     // O(n)

        System.out.println(list1);

        // Get Element      O(1)

        System.out.println(list1.get(2));

        // Remove Element   O(n)

        System.out.println(list1.remove(1));
        System.out.println(list1);
        
        // Set Element on index     O(n)

        System.out.println(list1.set(1, 200));
        System.out.println(list1);

        // Contains Element

        System.out.println(list1.contains(2000));

** Size of ArrayList
------------------------

-> The size of an ArrayList in Java refers to the number of elements currently stored in it, not its capacity. You can get this using the size() method.

-> .size()


    * Example :
    ------------

        System.out.println(list1.size());

        // Using size run the loop in ArrayList

        for(int i=0; i<list1.size(); i++) {
            System.out.print(list1.get(i) + " ");
        }
        System.out.println();
